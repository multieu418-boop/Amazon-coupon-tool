这是一个专门为亚马逊运营设计的自动化工具，用于快速生成 Coupon 提报文件并自动解析修复亚马逊的回传报错。

&nbsp;🌟 核心功能

一.阶段 1：生成提报 - 自动读取 Excel 模板规则，动态生成输入表单，支持 ASIN 批量格式化。

二. 阶段 2：报错修复 - 自动解析亚马逊报错文件中的批注，结合 All Listing 实时价格计算最大合规折扣。

三. 格式零破坏 - 采用 Openpyxl 引擎，确保输出的文件保留原始模板的所有下拉选项和隐藏格式。

&nbsp;🛠️ 使用前准备

在使用本工具前，请确保你手头有以下文件：

1\.ALL Listing Report: 从亚马逊后台导出的全量库存报告（用于获取 Price）。

2\.Coupon 提报模板: 亚马逊后台下载的空白 .xlsx 提报模板。

&nbsp;🚀 部署说明

本项目已配置为可直接在 Streamlit Cloud 部署：

1\. 将代码推送到 GitHub 私有仓库。

2\. 在 Streamlit Cloud 关联此仓库。

3\. 部署成功后即可通过网址在线访问。

&nbsp;⚠️ 注意事项

ASIN 数量：单次提报请确保子 ASIN 在 15-2900 个之间。

语言锁定：下拉框请直接选择模板原有的中文/英文选项，切勿修改。

数据安全：本工具在内存中处理数据，不进行任何云端存储。
